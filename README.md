# syntetic_data
Реализация генерации синтетических данных

generate_data3 - основной файл генерации и настройки

check_stat - проверка распределения

transactions_extended - файл с записями транзакций

precomputed - предрассчитанные средние данные

heatmap - данные кол-ва транзакций для тепловой карты активности
